# Arduino Obstacle Detection System

## Overview
This project is an Obstacle Detection System built using Arduino Uno and an IR Sensor. The system detects nearby obstacles and provides instant alerts through a buzzer and LED, making it useful for robotics, automation, and safety applications.

## Components Used
- Arduino Uno
- IR Sensor Module
- LED
- Buzzer
- Breadboard
- Jumper Wires

## Working Principle
1. The IR sensor continuously scans for nearby objects.
2. When an obstacle is detected, the sensor sends a signal to the Arduino Uno.
3. The Arduino processes the input and activates the LED and buzzer.
4. When no obstacle is detected, the alert system remains inactive.
