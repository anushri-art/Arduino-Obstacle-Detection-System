# Arduino Obstacle Detection System

## Overview
The Arduino Obstacle Detection System is a simple and effective embedded systems project developed using Arduino Uno and an IR Sensor. The system is designed to detect nearby obstacles and provide instant alerts through a buzzer and LED. This project demonstrates the practical application of sensor interfacing, microcontroller programming, and electronic circuit design.

## Components Used
- Arduino Uno
- IR Sensor Module
- LED
- Buzzer
- Breadboard
- Jumper Wires

## Working Principle
1. The IR sensor continuously monitors the surrounding area for obstacles.
2. When an object enters the sensor's detection range, the IR sensor sends a signal to the Arduino Uno.
3. The Arduino processes the signal and activates the LED and buzzer.
4. The LED provides a visual indication, while the buzzer provides an audible alert.
5. When no obstacle is detected, the alert system remains inactive.
